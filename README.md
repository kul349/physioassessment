## 📖 What is PhysioTest?

When a therapist performs a **Lachman Test** or a **Hawkins-Kennedy**, most patients are left wondering — *what just happened, and what does this mean for my recovery?*

**PhysioTest** pulls back the curtain. It's a free, open educational library that translates complex orthopedic and neurological assessments into clear, patient-friendly guides. Whether you're a patient preparing for your next appointment, a student, or a clinician needing a quick reference, PhysioTest has you covered.

> 🌐 **Live Site:** [https://physioassessment.vercel.app/](https://physioassessment.vercel.app/)

### 🧑‍⚕️ Patient Education
- **Assessment Stages** — a clear walkthrough of what happens during a full physio evaluation
- **Results Decoder** — what a "positive" or "negative" result actually means in plain language
- **Doctor Tips** — questions to ask your therapist about your results
- **Safety Info** — red flags and when to seek immediate medical attention

### 📄 Free Downloadable Guide
*The Patient's Guide to Physical Therapy* (PDF v2.4) covers:
- Comprehensive assessment checklists
- Self-screening techniques you can do at home
- How to prepare for your first visit
- Recovery timeline expectations

All content is reviewed by licensed physical therapy professionals and follows standardized clinical protocols.

## 🚀 Getting Started (Local Development)

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/physio-tests-app.git

# 2. Navigate into the project
cd physio-tests-app

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
